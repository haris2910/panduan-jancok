Langkah 1: Setup Webkus

Persiapan Wajib Sebelum Menyalakan Mesin Jancok Engine

Kenapa harus Setup Webkus dulu, Bosku?

Jancok Engine membaca seluruh data katalog produk langsung dari toko Webkus Bosku. Kalau tokonya masih kosong, produk belum ada, atau URL-nya salah, bot Telegram tidak akan bisa menampilkan menu belanja secara otomatis. Ikuti langkah di bawah ini dengan teliti!

Tahap A: Pendaftaran & Profil Toko

  1. Buka website resmi Webkus di browser Anda: https://www.webkus.com/
  2. Lakukan registrasi dengan mengeklik tombol Buat Toko Sekarang - Gratis.
  3. Pada tampilan dashboard Webkus, klik menu UBAH WEBSITE.
  4. Masukkan Nama Toko, Deskripsi Toko, Logo, dan URL Kontak. Kosongkan bagian Webhook URL, kemudian klik Simpan Perubahan.

Tahap B: Pembuatan Kategori & Produk

  1. Lanjut ke proses isi kategori produk. Klik menu Kategori.
  2. Masukkan nama kategori (contoh: KLIK INDOMARET) lalu klik tombol Buat Kategori.
  3. Lanjut ke proses pembuatan produk. Klik menu Produk, lalu pilih kategori di kanan atas yang sudah dibuat tadi, dan klik Buat Produk.
  4. Isi data produk: Nama produk, Kategori, Foto produk (bebas), dan Harga. Kosongkan bagian diskon jika memang tidak ada diskon.
  5. ⚠️ WAJIB PERHATIKAN: Pada pilihan jenis produk, pastikan Anda memilih UNIK (Jangan pilih DUPLICAT).
  6. Masukkan deskripsi produk, lalu klik BUAT PRODUK untuk menyimpan.

Tahap C: Pengisian Stok (Import CSV)

  1. Masih di menu Produk, silakan cari tulisan Stock dan klik tanda + (Tambah) di bawahnya.
  2. Masukkan Syarat dan Ketentuan (SnK) untuk produk tersebut di kolom yang disediakan.
  3. Siapkan file data akun/produk Anda. Jika file awal Anda berformat teks (misal: klik.txt), ubah nama ekstensi file tersebut menjadi klik.csv.
  4. Klik tombol Import CSV, lalu masukkan file produk .csv yang sudah disiapkan tadi.
  5. Proses selesai (Done)! Stok akan otomatis terisi.

🎉 Toko Webkus Sudah Siap! Setelah produk, harga, dan stok CSV berhasil diatur, Bosku siap untuk mengumpulkan data Bot Telegramnya!

Langkah 2: Data Bot & Script

Cara Mendapatkan BASE_URL, Token & ID Admin untuk Script

Di dalam script bot.js, ada 3 data wajib yang HARUS Bosku isi di bagian paling atas. Jika salah satu kosong atau salah, bot tidak akan jalan. Berikut cara mencari ketiganya:

A. Cara Mendapatkan BASE_URL (Link Webkus)

Ini adalah alamat toko Bosku agar bot tahu kemana ia harus mengambil data produk.

  1. Buka dashboard Webkus Bosku.
  2. Klik menu Ganti Nama Domain di sebelah kiri.
  3. Silakan edit nama subdomain sesuai nama toko Bosku, lalu klik Simpan Perubahan.
  4. Salin (Copy) link domain tersebut. (Contoh: https://toko-bosku.webkus.com)
  5. INGAT: Jangan pakai garis miring (/) di bagian paling akhir link!

B. Cara Mendapatkan BOT_TOKEN

Token ini ibarat "nyawa" agar script bisa mengendalikan bot Bosku.

  1. Buka aplikasi Telegram, ketik @BotFather di kolom pencarian, lalu klik Start.
  2. Ketik perintah /newbot dan kirim.
  3. BotFather akan meminta nama bot. Masukkan nama toko Bosku (Contoh: Jancok Store).
  4. Selanjutnya, masukkan Username Bot. Username ini WAJIB diakhiri dengan kata `bot` (Contoh: jancokstore_bot atau JancokStoreBot).
  5. Jika berhasil, BotFather akan mengirimkan pesan panjang. Cari tulisan Use this token to access the HTTP API:
  6. Salin (Copy) kode token panjang tersebut (Contoh: 1234567890:ABCdefGhIJKlmNoPQRstUVwxyZ).

C. Cara Mendapatkan ADMIN_ID

ID ini digunakan agar fitur rahasia (seperti Broadcast) hanya bisa diakses oleh Bosku saja sebagai Admin.

  1. Buka aplikasi Telegram, ketik @userinfobot di kolom pencarian, lalu klik Start.
  2. Bot akan langsung membalas dengan detail profil Bosku.
  3. Cari baris yang bertuliskan Id: (biasanya berupa deretan angka panjang).
  4. Salin (Copy) angka tersebut (Contoh: 8551325253).

✅ Persiapan Selesai! Sekarang Bosku sudah mengumpulkan BASE_URL, BOT_TOKEN, dan ADMIN_ID. Bosku sudah siap 100% untuk masuk ke tahap instalasi script di server!

Auto-Payment & Delivery

🛡️

Smart Cache (Anti-Limit 429)

⌨️

Input Qty Manual Bersih

Pembeli bisa ketik angka borongan (misal: 100 pcs). Chat otomatis dibersihkan agar riwayat Telegram tetap rapi.

📣

Broadcast Admin Pro

Admin bisa menyebarkan promosi (Foto, Video, Suara, File) dengan aman tanpa resiko akun bot di-banned.

🤖

Global Error Shield

Tameng gaib yang menelan semua error jaringan secara diam-diam. Log server Bosku dijamin bersih dari tulisan error merah.

📂 File Database Otomatis (Auto-Save)

Setelah bot dijalankan pertama kali, jangan kaget jika tiba-tiba muncul file-file baru di dalam folder bot Bosku. Ini adalah file database lokal yang dibuat secara otomatis oleh sistem:

  • users.txt
    Menyimpan daftar ID dan Nama pengguna yang pernah menggunakan bot. File ini dibaca saat Bosku melakukan Broadcast pesan.
  • sessions.json
    Menyimpan status keranjang belanja sementara dari setiap pembeli, agar pesanan mereka tidak hilang jika bot direstart.
  • history.json
    Menyimpan riwayat transaksi terakhir dari masing-masing pembeli (termasuk menyimpan data akun yang sudah dibeli).
  • testimoni.json
    Menyimpan ulasan, pesan, dan rating (bintang) yang dikirimkan oleh pembeli setelah pesanan selesai.

Instalasi di VPS (Linux)

Sangat Disarankan - Untuk Performa 24/7 Terbaik dan Super Ngebut

💡 Catatan Penting:

Panduan ini dikhususkan untuk VPS dengan OS Ubuntu atau Debian. Pastikan Bosku sudah login ke VPS menggunakan aplikasi SSH (seperti Termius di HP, PuTTY di PC, atau Terminal bawaan).

1. Persiapan Sistem & Install Mesin Utama (Node.js)

Kita harus memastikan server dalam kondisi terbaru dan menginstal Node.js versi 20 (LTS) sebagai "mesin" jalannya bot.

  • Ketik/Copy perintah di bawah ini satu per satu.
  • Tekan [ENTER] setiap selesai satu baris.
  • Jika ada pertanyaan (Y/n) saat proses instalasi, ketik y lalu Enter.
Terminal VPS (SSH) 
sudo apt update && sudo apt upgrade -y
sudo apt install curl -y
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt install -y nodejs

2. Buat "Gudang" Bot & Install Modul

Kita buat folder khusus agar file bot tidak berantakan, lalu mengunduh modul pendukung (Telegraf & Axios).

Terminal VPS (SSH) 
mkdir jancok-store && cd jancok-store
npm install telegraf axios

3. Masukkan & Edit Script Bot

Sekarang saatnya memasukkan kode bot sakti V104 ke dalam server.

  1. Buka teks editor Nano dengan mengetik: 
    nano bot.js
  2. Layar terminal akan berubah menjadi hitam kosong. PASTE (Tempel) seluruh kode script bot.js ke layar tersebut (Gunakan Klik Kanan jika di PuTTY PC, atau Long Press -> Paste di Termius HP).
  3. WAJIB: Gunakan tombol panah di keyboard untuk mencari dan mengubah bagian BOT_TOKEN, ADMIN_ID, dan BASE_URL sesuai data toko Bosku.
  4. Cara Save (Menyimpan):
    • Tekan tombol CTRL + X secara bersamaan.
    • Akan muncul pertanyaan di bawah "Save modified buffer?", tekan tombol Y.
    • Terakhir, tekan ENTER.

4. Menjalankan Bot 24 Jam (Pakai PM2)

Agar bot Bosku jalan otomatis di background dan langsung nyala sendiri ketika VPS di-restart (reboot), kita wajib pakai PM2.

Terminal VPS (SSH) 
sudo npm install -g pm2
pm2 start bot.js --name "jancok-store"
pm2 save
pm2 startup

*Catatan: Saat mengeksekusi pm2 startup, VPS biasanya akan memunculkan sebuah perintah teks panjang di layar. Bosku harus meng-COPY perintah panjang tersebut, lalu PASTE lagi di terminal, dan tekan ENTER.

💡 Perintah Pantau (Monitoring) PM2:

Gunakan perintah ini kapan saja untuk mengontrol bot Bosku di VPS:

  • Melihat pergerakan bot (siapa yang beli): pm2 logs (Untuk keluar layar log, tekan CTRL + C)
  • Restart Bot (kalau habis ganti/edit kode): pm2 restart jancok-store
  • Matikan Bot sementara: pm2 stop jancok-store
  • Melihat status bot (hijau/merah): pm2 list
  • Membersihkan layar log dari error lama: pm2 flush

Instalasi di Termux (Android)

Praktis jalankan bot hanya bermodal HP Android tanpa perlu PC/VPS.

⚠️ PERINGATAN KERAS:

Jangan pernah mengunduh aplikasi Termux dari Google Play Store karena versinya sudah mati dan usang (pasti akan error). Unduhlah aplikasi Termux dari F-Droid atau langsung dari rilis GitHub resminya.

1. Update Sistem & Izin Akses Penyimpanan

Pertama kali buka Termux, kita harus update paket dasarnya dan memberi izin agar Termux bisa menyimpan database pesanan di HP Bosku.

Aplikasi Termux 
pkg update && pkg upgrade -y
termux-setup-storage

*Jika saat loading muncul pertanyaan (Y/n), ketik y lalu Enter.

*Saat menjalankan termux-setup-storage, akan muncul pop-up di layar HP. Pilih "Izinkan" / "Allow".

2. Install Mesin Node.js & Modul Utama

Install Node.js sebagai mesin bot, aplikasi nano untuk edit teks, lalu buat folder jancok-store.

Aplikasi Termux 
pkg install nodejs nano -y
mkdir jancok-store && cd jancok-store
npm install telegraf axios

3. Setting Kode Bot (Wajib Teliti)

Buka layar editor teks untuk memasukkan kode Jancok Engine.

Aplikasi Termux 
nano bot.js
  1. Layar akan berubah hitam. Paste seluruh kode script yang sudah diberikan Admin.
  2. Geser menggunakan kursor/keyboard HP ke bagian paling atas. Ubah data BOT_TOKEN, ADMIN_ID, dan BASE_URL.
  3. Cara Save di Termux:
    • Tekan tombol CTRL (biasanya ada di barisan menu keyboard ekstra di atas keyboard HP).
    • Tekan huruf X.
    • Tekan huruf Y (untuk konfirmasi Yes).
    • Tekan ENTER.

4. Hidupkan Bot Pakai PM2 (Anti-Mati)

Biar bot nggak gampang mati saat aplikasi Termux ditutup atau HP layarnya mati, kita gunakan PM2 dan Wake-Lock.

Aplikasi Termux 
termux-wake-lock
npm install -g pm2
pm2 start bot.js --name "jancok-store"
pm2 logs

*Jika di log (pm2 logs) muncul tulisan "JANCOK ENGINE READY", artinya bot sudah sukses menyala! Tekan CTRL lalu C untuk keluar dari log.

🔄 JIKA HP MATI / DI-RESTART:

Karena Termux ada di HP, kalau HP Bosku mati/reboot, PM2 akan ikut mati. Bosku harus menyalakannya manual lagi. Buka aplikasi Termux lalu ketik:
cd jancok-store (lalu Enter)
pm2 start bot.js --name "jancok-store" (lalu Enter)

Instalasi di CMD (Windows/RDP)

Cocok untuk menjalankan bot di PC, Laptop, atau Virtual Desktop (RDP) Windows.

1. Download & Install Node.js

Kalau di Windows, kita tidak perlu pakai kode untuk install Node.js. Cukup klik-klik saja seperti install game/aplikasi biasa.

  1. Buka Browser (Chrome/Edge), lalu kunjungi website: https://nodejs.org/
  2. Download versi yang bertuliskan LTS (Recommended for most users). (Contoh: V20.x.x LTS).
  3. Buka file yang di-download (.msi), lalu install dengan mengeklik Next -> Accept -> Next -> Finish.

2. Siapkan Folder & File Script

Kita buatkan rumah untuk file botnya.

  1. Buka File Explorer (ikon folder kuning di bawah layar).
  2. Buat folder baru bebas dimana saja, misal di Drive D:, kasih nama foldernya jancok-store.
  3. Masuk ke dalam folder tersebut.
  4. Klik Kanan di area kosong ➔ New ➔ Text Document.
  5. Ubah nama file tersebut menjadi bot.js.
    (PENTING: Pastikan ekstensi .txt di akhir terhapus. Jika tidak melihat ekstensi .txt, klik menu 'View' di atas File Explorer, centang 'File name extensions').
  6. Klik Kanan file bot.js ➔ pilih Open With ➔ Notepad.
  7. Paste seluruh kode Jancok Engine ke dalam Notepad.
  8. Ubah konfigurasi BOT_TOKEN, ADMIN_ID, dan BASE_URL di bagian paling atas kode.
  9. Tekan CTRL + S untuk menyimpan (Save), lalu tutup Notepad.

3. Buka CMD & Install Library

Ini adalah trik cepat membuka CMD yang langsung mengarah ke folder bot Bosku.

  1. Kembali ke File Explorer, di dalam folder jancok-store tadi.
  2. Klik pada kotak Address Bar (kotak panjang yang menampilkan lokasi folder di bagian atas).
  3. Hapus semua tulisan di kotak tersebut, ketik cmd, lalu tekan ENTER. Jendela CMD hitam akan langsung terbuka!
  4. Di layar hitam CMD, ketik perintah ini untuk menginstal modul pendukung, lalu tekan Enter: 
    npm install telegraf axios

4. Jalanin Mesin & Penjaga PM2

Agar bot tidak otomatis mati jika jendela CMD tidak sengaja tertutup (close), kita wajib memasang PM2.

  1. Di jendela CMD yang sama, install PM2: 
    npm install pm2 -g
  2. Mulai jalankan bot secara permanen di background Windows: 
    pm2 start bot.js --name "jancok-store"
  3. Cek log untuk memastikan bot menyala tanpa masalah: 
    pm2 logs

🎉 STATUS: BERHASIL!

Bot Bosku sekarang sudah siap menerima pesanan 24 jam nonstop selama PC/RDP terkoneksi dengan jaringan internet. Gunakan pm2 stop jancok-store jika ingin mematikan bot sewaktu-waktu.

Tanya Jawab (FAQ)

Solusi cepat untuk masalah yang sering ditanyakan pembeli.

Bagaimana cara merestart bot setelah saya mengedit kode bot.js?

Buka terminal/CMD di dalam folder bot Anda, lalu ketik perintah pm2 restart jancok-store. Perubahan kode Anda akan langsung aktif tanpa mematikan bot secara total.

Kenapa bot saya tidak mau membalas pesan / klik menu di Telegram?

Coba cek log bot Anda dengan mengetik pm2 logs. Pastikan tidak ada pesan error berwarna merah. Jika bot mati, cek kembali apakah BOT_TOKEN yang Anda masukkan sudah benar dan tidak ada spasi kosong di depan/belakang token.

Bagaimana cara mengubah harga atau menambah produk baru?

Kabar baiknya, Anda tidak perlu mengedit kode bot sama sekali! Silakan login ke dashboard website Webkus Anda, lalu edit harga, deskripsi, atau tambah produk dari sana. Bot Telegram Anda akan otomatis membaca data Webkus terbaru secara real-time saat pelanggan menekan tombol Belanja.

Muncul pesan "Checkout Gagal. Server sibuk" saat pelanggan membeli?

Ini bukan error dari script bot, melainkan server Webkus sedang penuh/down (mengalami 504 Gateway Timeout). Bot ini sudah dilengkapi fitur Auto-Retry. Minta pelanggan menunggu sebentar dan mencoba checkout lagi dalam 1-2 menit ke depan.

Saya ingin mematikan bot sementara waktu, bagaimana caranya?

Buka terminal/CMD, lalu ketik perintah pm2 stop jancok-store. Jika ingin menyalakannya kembali, cukup ketik pm2 start jancok-store.
